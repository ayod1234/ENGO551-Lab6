ENGO651/551 - Lab 6 Assignment

Objectives

  • Gain experience with line simplification

Overview

In this lab, you’ll learn about using Turf.js to add line simplification to your web mapping applications. You will be allowing a user to draw a polyline. When a user clicks on the “simplify button,” the map will show a simplified version of the polyline drawn by the user in a different colour.

Getting Started

Turf.js

Turf.js has a Simplify function you can use. In fact, it is based on Simplify.js.

Requirements

Alright, here are the requirements:

  1. Map: When a user visits the site, the user should be able to draw a polyline on the map.
  2. Analysis: When a user clicks on a button on the front end, the map shows a simplified line of the polyline drawn by the user. There should be another button on the front end so that the user can remove the polyline from the map and draw a new polyline.
  3. You can use any mapping library of your choice.
